# WriteFlow AI

AI-powered content generation platform built with Next.js.

## Features

- AI content generation
- Authentication with Clerk
- Supabase database history
- Markdown rendering
- PDF download
- Responsive dashboard UI

## Tech Stack

- Next.js
- TypeScript
- Tailwind CSS
- Clerk Auth
- Supabase
- Groq AI
